Position

  • Static - Statinė
  • Relative - Realiatyvioji
  • Absolute - Absoliučioji
  • Fixed - Fiksuotoji
  • Sticky - Lipnioji

Naudojant bet kurią apart Static pozicijos, gauname prieigą prie 5 naujų stilių: top, left, right, bottom ir z-index.
Top, left, right, bottom nurodo per kokį atstumą elementas turi būti atitrauktas.
Z-index stilius nurodo elemento prioritetą sluoksniuose (esant persidengimui nurodo kuris elementas bus viršuje ar apačioje).

Statinė pozicija - nieko įpatingo nedaro, net nesuteikia galimybės naudoti top/left/rigt/bottom/z-index. Labiau skirtas panaikinti kokią nors savybę.
