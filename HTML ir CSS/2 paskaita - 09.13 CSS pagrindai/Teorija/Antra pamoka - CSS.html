CSS rašymo būdai

  • Inline - linijinis rašomas tag'o viduje naudojant style atributiką
  • Internal - vidinis, rašomas style tag'o viduje (head dalyje)
  • External - Išorinis rašomas atskirame css faile, kurį reikia prijungti prie html naudojant link'o tag'ą

CSS selektor'iai

  • element - Pasirenkamas elementas pagal jo pavadinimą (p{})
  • id - Sukuriamas susigalvotas id pavadinimas ir priskiriamas vienam elementui (#idPavadinimas{})
  • class - Sukuriamas susigalvotas klasės pavadinimas ir priskiriamas bet kokiam kiekiui elementų (.klasėsVardas{})

CSS Stiliai

  • color - skirtas pakeisti teksto spalvą (color: red)
  • font-size - skirtas pakeisti teksto dydį (font-size: 20px)
  • background-color - skirtas pakeisti fono spalvą (background-color: aqua)
  • font-weight - skirtas pakeisti teksto storumą (font-weigt: bold)
  • line-height - skirtas pakeisti teksto eilutės aukštį (line-height: 20px)
  • font-family - skirtas pakeisti teksto šriftą (font-family: monospace)
  • border - skirtas nurodyti rėmelio storį+tipą+spalvą (border: 5px solid black)
  • border-radius - skirtas apvalinti rėmelio kampus (border-radius: 10px)
  • text-aligh - skirtas lygiuoti tekstą (text-align: center)
  • font-style - skirtas suteikti tekstui stilių (font-style: italic)

Spalvos

  • Spalvos vardo naudojimas, yra 140 spalvų vardų (red)
  • HSL naudojimas (niekas nenaudoja)
  • HEX naudojimas. Naudojama taip pat gausiai kaip ir RGB (#FF000)
  • RGB naudojimas. Naudojama taip pat gausiai kaip ir HEX ((rgb)255,0,0))

Permatomos spalvos

  • HSL permatomumas nurodomas nuo 0 (pilnai permatomas) iki 1 (pilnai matomas). (hsla(0,100%,50%,1))
  • HEX permatomumas nurodomas nuo 00 (pilnai permatomas) iki FF (pilnai matomas)(#FF0000FF).
  • RGB permatomumas nurodomas nuo 0 (pilnai permatomas) iki 1 (pilnai matomas)(rgba(255, 0, 0, 1)).

HEX ir RGB

RGB - RedBlueGreen. Kiekvienos spalvos kiekio nurodyti nuo 0(nesuteikiama spalvos) iki 255 (suteikiama spalvos maksimaliai).
