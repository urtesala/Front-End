Parent ir child elements

Elementas, kuris savo viduje turi kitus elementus yra TĖVINIS (PARENT) elementas.
Elementas, kuris yra kažkokio elemento viduje yra vadinamas VAIKINIS (CHILD) elementas.

Elementas gali turėti vieną vienintelį tėvinį (parent) elementą.
Elementas gali turėti daugelį vaikinių (child) elementų,

  • Pirmas
  • Antras
  • Trečias
    • Trečio pirmas
    • Trečio antras
    • Trečio trečias
    • Trečio ketvirtas
  • Ketvirtas

CSS selectors

Visus CSS selektorius galima rasti

  • element - tago vardą
  • .class - tagui suteiktą klasę
  • - tagui suteiktas id
  • .class1.class2 - tagui suteiktos abi (arba daugiau) klasės
  • .class3 .class4 - tagui suteikia stilius, jei jis turi class4 klasę ir yra class3 palikuonis (desendent)
  • * - parenka viską
  • element1, element2 - išvardintus elementus ir kableliu atskirtus elementus
  • element1 > element2 / element2 - element1 vaikinis elementas element2 / element2, kurio tėvinis elementas yra element1
  • a:link - nelankytos nuorodos
  • a:visited - lankytos nuorodos
  • a:active - aktyvi nuoroda (paspaudimo momentas)
  • element:hover - elementas ant kurio užvesta pelė
